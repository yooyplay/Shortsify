Installation steps

1

Open the DMG and drag to Applications

Double-click Shortsify.dmg, then drag Shortsify.app into the Applications folder shown in the window.

2

macOS will show a security warning — here's how to fix it

Because Shortsify isn't sold through the Mac App Store, macOS shows this on first launch:

!
"Shortsify" Not Opened
Apple could not verify "Shortsify" is free of malware that may harm your Mac or compromise your privacy.
⚠ Don't click "Move to Trash"

This warning appears for any app not distributed through Apple's store — it doesn't mean the app is harmful. Shortsify is open source and safe.

To bypass it, open Terminal (⌘ Space → type Terminal → press Enter) and run:

xattr -cr /Applications/Shortsify.app

Then double-click Shortsify normally. You only need to do this once.

3

Launch and go

Double-click Shortsify in Applications. On first launch it automatically downloads ffmpeg in the background — no extra setup. Just drop a video and convert.

💡 Tip: Want to skip the security warning entirely? Use Homebrew instead — installs with no warnings and keeps itself updated.

🍺 What is Homebrew?

Homebrew is a free, trusted package manager used by millions of Mac developers — think of it as an app store you control from the Terminal. It downloads, verifies, and installs apps safely. It also handles updates automatically. No Gatekeeper warnings when installing through Homebrew.

Install via Homebrew

1

Install Homebrew (skip if you already have it)

Open Terminal (⌘ Space → type Terminal) and run:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
2

Install Shortsify — three commands

brew tap yooyplay/tap
brew trust yooyplay/tap
brew install --cask shortsify

Homebrew downloads the app, verifies its integrity, and installs it — no security warnings.

3

Update anytime with one command

brew upgrade --cask shortsify
