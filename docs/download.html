Installation

1

Open the DMG and drag to Applications

Double-click Shortsify.dmg, then drag Shortsify into the Applications folder shown in the window.

2

Bypass Gatekeeper (one-time, 10 seconds)

Because Shortsify isn't signed with an Apple certificate, macOS blocks the first launch. Open Terminal (press ⌘Space, type Terminal) and run:

xattr -cr /Applications/Shortsify.app

Then double-click the app normally. You only need to do this once.

3

That's it — no setup needed

On first launch, Shortsify automatically downloads everything it needs. Just drop a video and go.
