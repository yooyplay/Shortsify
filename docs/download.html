Installation

1

Install ffmpeg

Shortsify uses ffmpeg under the hood. If you don't have it:


brew install ffmpeg

Don't have Homebrew? Install it here ↗

2

Unzip and move to Applications

Double-click the downloaded zip. Then drag Shortsify.app into your /Applications folder.

3

Bypass the "unrecognized developer" warning

Because Shortsify isn't signed with an Apple Developer certificate, macOS will block the first launch. Here's how to open it:


Option A (easiest): Right-click Shortsify.app → Open → click Open in the dialog. You only need to do this once.


Option B: Go to System Settings → Privacy & Security, scroll down, and click "Open Anyway" next to the Shortsify message.


Option C (Terminal):


xattr -cr /Applications/Shortsify.app

Then open normally.

4

You're done!

Drop any video onto the window, pick your output format(s), and Shortsify handles the rest.
